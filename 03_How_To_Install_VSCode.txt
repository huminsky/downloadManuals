How To Install VSCode on Jetson Nano:
===========================================================================
cd

(1) Download the deb file:
wget https://github.com/toolboc/vscode/releases/download/1.32.3/code-oss_1.32.3-arm64.deb

(2) Install:
sudo dpkg -i code-oss_1.32.3-arm64.deb

(3) Excute:
code-oss
